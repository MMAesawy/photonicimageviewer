Photonic Image Viewer is a lightweight image viewing program designed primarily for the Microsoft Windows operating system.

Photonic Image Viewer was made by Mohamed Aesawy as a personal hobbyist project. The software is open source and its code can be viewed or modified at your pleasure at its GitHub repository. It was made using JavaFX developed by the Oracle Corporation, and uses the Metadata-Extractor library developed by Drew Noakes.

You may contact the developer —Mohamed Aesawy— at m.m.aesawy@gmail.com. Send your criticisms, compliments, or comments or just say hi!
